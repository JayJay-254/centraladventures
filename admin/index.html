Central Adventures — Admin

Use this dashboard to inspect and manage Firestore documents (trips, bookings).

Admin Sign-in

Sign in with Firebase Email/Password (recommended). The app must be initialized with your Firebase web config first.

Before using: Add your Firebase web config on this page (see README) as `window.ADMIN_FIREBASE_CONFIG` in DevTools, or create `admin/config.js` (not committed).
